## 概要

本ツールは、正規表現をテストするための Windows 向けユーティリティです。  
.NET Framework の正規表現規則に従って動作しますが、Perl 互換に近いため、他の用途にも流用可能です。

---

## インストール

アーカイブ内のファイルを、任意のディレクトリに解凍してください。

---

## 動作環境

> **注意**  
> バージョン 1.1 以降は **.NET Runtime 2.0** が必要です。

- .NET Runtime 2.0
- 十分なメモリおよび CPU

**参考（開発環境）**
- OS: Windows XP Pro SP2  
- CPU: Pentium4 1.7GHz  
- Memory: 768MB  
- 開発環境: Visual Studio .NET 2005 / C#  
- .NET Framework & CLR 2.0

---

## 使い方

1. 画面最上部に **テスト対象文字列** を入力します。
2. 「正規表現」欄に正規表現を入力し、**検索**ボタンを押します。
3. 結果はツリー形式で表示されます。
   - 各ノードはマッチした部分文字列を表します。
   - グループ化した場合は、子ノードとして表示されます。
4. ツリーをクリックすると、該当箇所がテスト対象文字列内で反転表示されます。
5. 同ディレクトリにある `retsample.txt` は、右下のメモ欄に表示されます。
6. 置換表現を入力して **置換** ボタンを押すと、文字列が置換されます。
7. **元に戻す** ボタンで変更を取り消せます。

---

## オプション

- **大文字と小文字を区別しない**  
  アルファベットの大文字・小文字を区別せずにマッチします。

- **「.」が改行にもマッチする**  
  いわゆる *シングルラインモード* です。

- **「^ $」が各行ごとにマッチする**  
  いわゆる *マルチラインモード* です。
